Case: The Windrush Scandal

Never Heard of the Windrush Scandal?

  • “Windrush Generation”: migrant workers from Commonwealth countries invited to rebuild UK between 1948 and 1973
  • Policy changes towards a” hostile environment” for illegal immigrants from 2012
  • Since “Windrush Generation” needed no visa, they never immigrated per se (plus government destroyed legal records)
  • NGOs and journalists warned since 2013 that “Windrush Generation” were targeted despite living in UL for decades
  • March to May 2018: the scandal explodes into media storm and leads to policy changes and austing of Home Secretary Amber Rudd
  • striking about the case:
    • not a new problem
    • when the media discovered the issue a strong response throughout the hybrid media system
    • actual political consequences (not solved but personnel change)

The Puzzle: why is there still a public agenda when the media is fractured?

Windrush Scandal Media Storm (a Waves of Collective Attention?)

  • In Langer & Gruber (2021) we discuss the media storm that created pressure on the government
  • Dataset: Newspaper (N = 1,264), TV (245), Alternative media (263), petitions (15), Twitter (1,769,940)
  • Conclusion: media types reacted similarly – despite fierce battles for attention, shortening attention spans and fractured audiences
  • Conclusion 2: news media still at heart of conversation, especially shown by URL shares (theguardian.com: 32%, bbc.co.uk: 14%)

Still some open questions:

  1. Are media types really talking about the same thing?
  2. Who leads, who follows?

Text Embedding

  • Transforming text into fixed-length vectors
  • Capturing semantic meaning and structure
  • Large pretrained general purpose models readily available (e.g., BERT)
  • Useful for NLP tasks (e.g., machine learning, translation, (semantic) similarity)
  • Increase performance while decreasing computation

Relational Event Models

  • Inferential network analysis technique
  • Perfect for fine grained temporal data
  • Main question: how are events influenced by previous events and interactions in the network
  • Event = message (actor + concept + date time)
  • Influence: how does network of past events influence actors?
  • Exponential decay of past events -> more recent events = more important influences
